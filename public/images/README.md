# Imágenes del Sistema

## Imagen de Fondo

Coloca tu imagen de fondo en esta carpeta con el nombre: **`background.jpg`**

### Requisitos recomendados:
- **Formato**: JPG, PNG o WebP
- **Resolución**: Mínimo 1920x1080px (Full HD)
- **Tamaño**: Optimizado para web (máximo 2-3MB)
- **Nombre del archivo**: `background.jpg`

### Nota:
La imagen tendrá un overlay negro translúcido (70% de opacidad) aplicado automáticamente para asegurar que el contenido sea legible sobre cualquier imagen.

Si deseas ajustar la opacidad del overlay, edita el archivo `resources/css/custom-theme.css` y modifica la línea:

```css
background: rgba(0, 0, 0, 0.7); /* 0.7 = 70% de opacidad */
```

Donde:
- `0.5` = 50% de opacidad (más transparente)
- `0.7` = 70% de opacidad (recomendado)
- `0.8` = 80% de opacidad (más oscuro)
